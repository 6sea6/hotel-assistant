排名算法说明

自定义权重排名算法：

系统根据四个维度计算每家宾馆的综合得分，得分越高排名越靠前。

一、各维度归一化公式：

价格得分 = (maxPrice - 当前价格) / (maxPrice - minPrice)

评分得分 = 当前评分 / 5

距离得分 = (maxDistance - 当前距离) / (maxDistance - minDistance)

交通得分 = (maxTime - 当前时间) / (maxTime - minTime)

二、综合得分公式：

S = Wp × Sp + Ws × Ss + Wd × Sd + Wt × St

参数说明：

  • S：综合得分（0-1之间）
  • Wp：价格权重，Sp：价格得分
  • Ws：评分权重，Ss：评分得分
  • Wd：距离权重，Sd：距离得分
  • Wt：交通权重，St：交通得分

注：归一化确保各维度在0-1之间，消除量纲影响，使权重设置更有意义。
