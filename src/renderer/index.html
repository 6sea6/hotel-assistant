排名算法说明

自定义权重排名算法：

系统根据四个维度计算每家宾馆的综合得分，得分越高排名越靠前。

一、各维度归一化公式：

价格得分 = (maxPrice - 当前价格) / (maxPrice - minPrice)

评分得分 = 当前评分 / 5

距离得分 = (maxDistance - 当前距离) / (maxDistance - minDistance)

交通得分 = (maxTime - 当前时间) / (maxTime - minTime)

二、综合得分公式：

S = Wp × Sp + Ws × Ss + Wd × Sd + Wt × St

参数说明：

  • S：综合得分（0-1之间）
  • Wp：价格权重，Sp：价格得分
  • Ws：评分权重，Ss：评分得分
  • Wd：距离权重，Sd：距离得分
  • Wt：交通权重，St：交通得分

注：归一化确保各维度在0-1之间，消除量纲影响，使权重设置更有意义。

📖 应用说明书

🏨 宾馆管理

  • 添加宾馆：点击左侧"添加宾馆"按钮，填写宾馆信息后保存
  • 编辑宾馆：点击宾馆卡片上的"编辑"按钮，修改信息后保存
  • 补充交通信息：可填写最近地铁站名称、距离最近地铁站的距离与公交路线，地铁距离填 0 时界面会显示“无较近地铁站”
  • 原始房型：可单独记录平台原始展示房型名，卡片会显示在宾馆名称下方，便于区分标准房型和平台原文
  • 卡片删除：卡片视图中每个宾馆都带删除按钮，第一次点击进入黄色确认态，再点一次才会删除
  • 行式删除：行式视图请勾选左侧复选框后，使用右上角"删除选中"统一删除
  • 收藏宾馆：点击"收藏"按钮，宾馆会显示金色边框标识

📋 模板系统

  • 创建模板：点击"模板管理" → "新建模板"，设置常用的入住日期、目的地等信息
  • 模板列表：每个模板会显示目的地、入住日期、离店日期和房间人数，方便快速核对行程范围
  • 应用模板：添加宾馆时选择模板，自动填充预设信息
  • 模板关联：宾馆只会使用你显式选择或写入的模板，不再根据日期自动猜测模板

🔍 搜索筛选

  • 宾馆名称：程序会自动汇总当前已有宾馆名称，直接选择即可筛选
  • 价格排序：按总价格升序或降序排列
  • 携程评分：筛选4.0分或4.5分以上的宾馆
  • 收藏状态：筛选已收藏或未收藏的宾馆
  • 按模板筛选：显示使用特定模板的宾馆
  • 公共交通时间：筛选20/40/60/80/100分钟以内的宾馆
  • 交通补充信息：详情中可查看最近地铁站名称、最近地铁站距离与公交路线，地铁距离为 0 时显示“无较近地铁站”
  • 列表信息展示：卡片视图会在宾馆名称下显示原始房型，地址固定在左侧保留可读宽度，网址固定从右侧信息列起始位置开始显示并保持单行省略；若填写了公交路线，会在卡片信息区独占整行显示；行式视图不显示地址和公交路线，只在名称下补充原始房型
  • 最近地铁站筛选：支持筛选无、300米、500米、1km、1.5km 以内的宾馆
  • 网址展示：卡片中网址显示在地址右侧独立列，默认只显示首行并自动省略，但点击后仍会直接打开原网页

📊 排名算法

  • 自动排名：系统根据默认权重自动计算排名
  • 自定义权重：手动调整价格、评分、距离、交通的权重值
  • 排名规则：综合得分越高排名越靠前，前三名显示金色徽章

🎨 个性化与设置

  • 个性化：点击顶部“个性化”按钮，可切换龙猫蓝、甜柠青、小草绿、菠萝黄、橡木棕、云朵白、秋仪金、铁杆粉、葡萄紫、彩色十套主题，并管理应用图标
  • 采集偏好：可控制“3人模板时是否额外保留携程显示可住4人的房型”
  • 应用图标：支持在个性化面板中选择本地图片作为窗口图标；所选图片会自动复制到当前数据目录内，删除原图也不会让窗口图标失效，也可以一键恢复默认图标
  • 数据存储位置：查看和更改数据存储路径
  • 快捷链接：快速访问携程、飞猪官网
  • AI提示词：查看和编辑保护性提示词、数据填写指南、优化提示词
  • 开发者工具：按 F12 打开后会优先切到中文界面，便于调试页面结构和脚本

💾 数据管理

  • 导出数据：将所有宾馆、模板和设置导出为JSON文件备份；如果当前启用了自定义窗口图标，也会一起写进导出文件
  • 导入数据：从本应用自己导出的 JSON 恢复数据，可选择覆盖当前宾馆、模板和设置，或仅把宾馆与模板追加到现有数据中；追加模式会保留当前设置和窗口图标，并自动重排冲突 ID，避免模板关联串号；任一步失败时都会回滚
  • 导出排名图片：将当前排名列表导出为PNG图片；为避免文字重叠，图片中不显示宾馆地址和交通路线，总价与日均会合并到同一信息格上下显示，其余字段按每行三个信息格重排，最近地铁站以“距地铁”显示且仅展示距离

🤖 AI辅助功能

  • 保护性提示词：明确不能破坏数据结构、IPC 与模板同步逻辑
  • 数据填写指南：指导 AI 以结构化 JSON 方式稳定补全宾馆数据
  • AI优化提示词：强调先定位根因，再优化卡顿、体积、DevTools 调试体验和维护性

💡 使用技巧

  • 使用模板可以快速添加多个行程相似的宾馆
  • 手动添加或编辑宾馆时，模板只会按你当前表单里的选择保存，不会再被程序自动改绑
  • 收藏功能可以标记心仪的宾馆，方便对比
  • 列表右上角的刷新图标会重新拉取当前窗口内的宾馆、模板和设置数据，不需要重启应用
  • 需要批量删除时，先切到行式视图勾选多家宾馆，再使用右上角删除选中
  • 导出数据可以备份重要信息，防止数据丢失
  • 排名图片可以分享给朋友一起讨论
